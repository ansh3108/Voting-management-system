## Installation Instructions

1. Install XAMPP or any other local server environment.

2. Use a text editor of your choice (e.g., Notepad++, Sublime Text, VS Code).

3. Download the "Voting_Management_System" project files.

4. Extract the downloaded file.

5. Copy the "Voting_Management_System" folder.

6. Paste the folder into the root directory of your local server (e.g., `xampp/htdocs`).

7. Open PHPMyAdmin by visiting `http://localhost/phpmyadmin`.

8. Create a new database named `voting_db`.

9. Import the `voting_db.sql` file located in the SQL folder of the project.

10. Run the application by visiting `http://localhost/Voting-Management-System`.
